.. currentmodule:: moddb.boxes

ModDB Boxes
============
A 'Box' is a div tag which resides on the page but does not encompass the page. It is often on
the sidebar with for goal to provide meta data on the page it is attached too. Few of these will
be returned as individual objects, they are mostly used as attributes of Pages.
